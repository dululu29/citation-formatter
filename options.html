Manage Custom Abbreviations

Define your own abbreviations here in JSON format. Enter key-value pairs where the key is the **full publication name** (case-sensitive, exactly as it appears in the metadata if possible) and the value is your desired abbreviation.

Example: {"IEEE Transactions on Wireless Communications": "MyTWC", "Global Communications Conference": "MyGlobecom"}

These abbreviations will be used if 'Use Custom List' is selected for either Full Mode or PowerPoint Mode in the Configuration tab. Invalid JSON will not be saved.
